## How to use
1. Replace images in `assets/` and replace `assets/resume.pdf` with your PDF resume.
2. Edit links (LinkedIn/GitHub) in `index.html`.

## Deploy to GitHub Pages (free)
1. Create a repository `sonalisawwasher.github.io` on GitHub.
2. Push the folder contents to the repository (main branch).
3. GitHub Pages will publish at: https://sonalisawwasher.github.io

## Deploy to Netlify (drag & drop)
1. Zip the folder and drag it into Netlify Drop (https://app.netlify.com/drop)
2. It will deploy and give you a live URL.
